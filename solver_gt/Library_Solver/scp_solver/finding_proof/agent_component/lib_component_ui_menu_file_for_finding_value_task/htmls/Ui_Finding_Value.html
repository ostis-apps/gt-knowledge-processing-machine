Команда проверки истинности утверждения предназначена для проверки истинности поданного утверждения. Единственным аргументом команды является атомарная формула, истинность которой необходимо проверить. Результатом выполнения запроса является появление в памяти доказательства утверждения, соответствующего атомарной формуле.
